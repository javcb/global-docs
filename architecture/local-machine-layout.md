# Local machine layout

## Goal

Local folders should mirror GitHub locations closely enough to preserve a 1:1 mental model.

## Preferred root

Use a single root such as:

~/code/github/

## Layout

- ~/code/github/yourname/
- ~/code/github/yourname-templates/
- ~/code/github/yourname-staging/
- ~/code/github/yourname-archive/
- ~/code/github/javcb-ai/

## Rules

- Only keep actively used repos cloned locally.
- Remove dormant repos from the machine when not in active use.
- Re-clone on demand.
- Local folder names should match repo names.
