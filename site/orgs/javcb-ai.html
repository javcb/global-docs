Org: javcb-ai

Purpose

AI sandbox. Where coding agents can work freely without risk to other systems.

What belongs here

  • AI-generated prototypes
  • Autonomous agent experiments
  • End-to-end builds initiated by plain-English prompts
  • Construction estimator, mockups, Supabase experiments, etc.

What does not belong here

  • Core libraries that others depend on
  • Production code

Allowed languages

  • Any language allowed
  • Prefer Python/SQL for anything that might be promoted to prod
  • TypeScript/JavaScript/CSS freely used for UI prototypes

AI posture

  • AI tools have full write access here
  • Agents may create repos, commit, scaffold, and iterate
  • Output should be clearly labeled prototype vs production-candidate

Review before promotion

  • When a repo is ready for review, AI should update README with:

- What it does - What to test - What is incomplete or hardcoded

  • Owner reviews and decides whether to promote to staging or archive
