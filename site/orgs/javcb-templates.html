Org: javcb-templates

Purpose

Pristine read-only starting points. Templates are not active projects.

What belongs here

  • Purchased UI kits (Tailwind, component libraries)
  • Excel/Power Query starter templates
  • Python project starter structures
  • Reusable boilerplate repos

What does not belong here

  • Active work
  • Modified or customized instances of templates

Allowed languages

  • Any, based on template type

AI posture

  • AI tools should have read-only access
  • Templates should not be modified directly
  • To use a template: clone or copy into a new staging repo

Rules

  • All repos here should be archived (read-only on GitHub)
  • Include source/purchase details and license notes in each repo README
