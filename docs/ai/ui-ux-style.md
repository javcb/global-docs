# UI/UX style guidance

## Inspiration

Use design inspiration from top-tier product companies such as:
- Apple
- Netflix

## Principles

- clean hierarchy
- minimal clutter
- strong spacing and visual rhythm
- obvious calls to action
- restrained color use
- accessibility and readability first

## Prototype discipline

- distinguish prototype from production-ready implementation
- do not overstate completeness
- prefer realistic, usable flows over flashy mockups
