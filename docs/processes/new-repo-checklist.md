# New repo checklist

- what org/user should this live under?
- is this prod, staging, template, archive, or AI sandbox?
- what languages are allowed here?
- is this core logic or an adapter/workflow?
- should SQL live here or in sql-core?
- what secrets/config are required?
- what AI instruction stub should link back to global-docs?

## Prompt

Before creating a new repository, review this checklist with me: Which org/user should own it? Is it prod, staging, template, archive, or AI sandbox? What languages are allowed? Is this core logic or an adapter/workflow? Should SQL go here or in sql-core? What secrets/config are needed? What AI instruction stub should reference global-docs?
